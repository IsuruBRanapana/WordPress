Verge
=========

Verge has been designed for Sites Based on Pictures and Videos. But, its so flexible it can adapt to almost any other kind of site as well. It supports and relies on Featured Images. Has a Beautiful Navigation Bar, Featured Posts Section, Works with RT Slider and supports Multiple Blog Layouts. Sidebar is also configurable. And Yes, its Fully Responsive on all devices from smallets phones to bigger desktops.

Getting Started
---------------

How to Install the Theme?

1. Go to WP Admin > Appearance > Themes > Add New.
2. Search for "Verge".
3. Click on Install and Activate.

Alternatively,

1. Go to WP Admin > Appearance > Themes > Add New > Upload.
2. Upload the verge.zip file, in which you found this README.txt
3. Click on Install and Activate.

Setting it Up
-------------

This theme is built using Customizer. So to Configure theme settings like Featured Content Area, Sliders, Social Icons, etc from 
	
	Dashboard > Appearance > Customizer
